# 🎨 ارسم معاً - تطبيق الرسم الجماعي

## المميزات
- ✏️ فرش متنوعة: دائري، مربع، رش، خط، نيون، ماركر
- 🎨 عجلة ألوان مع إمكانية حفظ الألوان
- 😊 ستيكرات وإضافة صور
- 🏠 غرف خاصة بكود 6 أرقام (حد 4 أشخاص)
- 💬 دردشة داخل الغرفة تدعم العربية
- 🤝 نظام إضافة أصدقاء ودعوتهم
- 🖼️ حفظ اللوحة كصورة
- 📱 متوافق مع الهاتف والكمبيوتر
- 🔐 تسجيل دخول بدون إيميل (session)

## التثبيت

```bash
pip install flask flask-socketio eventlet
```

## التشغيل

```bash
python app.py
```

ثم افتح المتصفح على:
```
http://localhost:5000
```

## الاستخدام

1. **سجّل دخولك** باختيار اسم وأفاتار ولون
2. **أنشئ غرفة** أو **انضم بكود** موجود
3. **شارك الكود** مع أصدقائك (6 أرقام)
4. **ارسموا معاً** واستمتعوا!

## ملاحظة
- الغرف مؤقتة وتختفي عند إغلاق السيرفر
- الحد الأقصى للغرفة 4 أشخاص
